# AutoInspect Pro

AutoInspect Pro is a premium vehicle inspection report management system built with Streamlit, SQLite, ReportLab, and Pillow. It provides a modular workflow for capturing vehicle information, uploading photos, recording inspection findings, calculating a vehicle score, and generating a PDF report.

## Features
- Secure authentication with bcrypt
- Responsive dashboard with summary cards
- Vehicle information capture and validation
- Multi-image upload with compression
- Inspection modules for body shell, chassis, paint, exterior, interior, and more
- Inspection summary and scoring
- PDF report generation
- Report history and search

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
